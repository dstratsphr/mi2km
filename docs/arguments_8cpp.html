int get_arguments ( int  argc,
char *  argv[] 
)

Process the command line arguments. Set options and store args.

Check each argument. If it is an option apply the setting to the global option variable that's indicated or carry out the required command. Check for --help amd --version options first, since they terminate the program after output. Any strings that aren't supported options go in the global vector args.

Parameters
argcThe number of command line arguments.
argvA C array containing the command line arguments passed to the program by the user.
